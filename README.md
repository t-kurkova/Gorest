# Gorest

## Описание

https://gorest.co.in/ Данный сайт представляет собой сервис REST API для тестирования и прототипирования.
Реалистичные данные для пользователей, публикаций, комментариев и задач. Аутентификация с помощью Bearer-токена.
В качестве тестирования взят объект **Comments** и **User**
